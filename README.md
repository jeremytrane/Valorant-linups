# Valorant Lineups

![Valorant Lineups Logo](./images/logo.png)

Valorant Lineups is the ultimate destination for Valorant enthusiasts. Dive deep into our meticulously curated lineups, designed to maximize the potential of every agent's utility. Whether you're a seasoned veteran aiming to refine your strategy or a newcomer eager to grasp the nuances, our guides offer insights that cater to every skill level. Stay a step ahead, optimize your gameplay, and rise through the ranks with the power of knowledge. Let's redefine the meta, one lineup at a time.

## Features

- **Responsive Design**: The website is designed to be fully responsive, adapting seamlessly to different screen sizes.
  
- **Navigation Bar**: A minimalist navigation bar that links to the official Valorant website, the project's GitHub repository, and an About page.

- **Agent Cards**: Interactive cards showcasing different Valorant agents with their images and roles.

## Design

The design incorporates a dark theme with white text for better readability. Background images of Valorant characters add aesthetic appeal and relevance to the content.

### Typography
- Main font: `Roboto`
- Secondary font (for headings and specific elements): `Press Start 2P`

### Color Palette

- Text Color: `#fff`
- Background Gradient: `linear-gradient(rgba(0, 0, 0, 0.7), rgba(0, 0, 0, 0.7))`

### Layout

The layout is split into two main sections:

1. A description section which introduces the user to the purpose of the website.
2. A section showcasing different Valorant agents with their roles.

## Development

This project is developed using vanilla HTML and CSS. The CSS ensures a fluid and responsive design to cater to a variety of screen sizes.
